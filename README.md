CS50 Week 0 Scratch Project

Project Title
My CS50 week 0 Game project

Description
A fun interactive game where the player controls a character(dino) and tries to catch falling object.

Features
- Player can move left and right
- Objects fall randomly from the top
- Score increases when you catch objects
- Game ends after repeating 10 times.

How to Use
Click the green flag to start. Use arrow keys to move your character.
